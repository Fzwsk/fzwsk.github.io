Linux Server Hardening (Production Baseline)

Category: Linux

Environment: RHEL / CentOS / Ubuntu

Purpose: Initial security baseline before go-live

1️⃣ Update System


# RHEL / CentOS
dnf update -y

# Ubuntu
apt update && apt upgrade -y

2️⃣ Disable Root SSH Login


vi /etc/ssh/sshd_config

PermitRootLogin no
PasswordAuthentication no

systemctl restart sshd

3️⃣ Configure Firewall


# RHEL
firewall-cmd --permanent --add-service=http
firewall-cmd --reload

# Ubuntu
ufw allow 80
ufw enable

4️⃣ Fail2Ban (Optional)


dnf install fail2ban -y
systemctl enable fail2ban
systemctl start fail2ban

5️⃣ Time Sync


timedatectl set-ntp true
